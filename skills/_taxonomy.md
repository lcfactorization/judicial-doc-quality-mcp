# A系列异常分类体系

检测到的每个异常点必须映射到以下A系列分类之一：

| 编号 | 分类名称 | 描述 | 关联维度 | 严重度基数 | 双向适用 |
|------|---------|------|---------|-----------|---------|
| A1 | 关键证据未回应 | 对一方当事人提交的核心证据，裁判文书未予回应、评述或采信/排除说理 | D2,D3,D10,D15 | 0.85 | 是 |
| A2 | 事实认定跳跃 | 从证据到事实的推理链条存在断裂，关键中间环节缺失论证 | D3,D9 | 0.80 | 是 |
| A3 | 法律适用未解释 | 适用某一法条但未说明为何适用该法条而非另一法条，或未对法律概念进行界定 | D5,D6,D14 | 0.75 | 是 |
| A4 | 同类证据双重标准 | 对同类证据因提交方不同而采用不同的采信标准 | D2,D3 | 0.90 | 是 |
| A5 | 程序时间线异常 | 程序行为的时间顺序存在逆序、超期、加速等异常模式 | D1,D10,D13 | 0.70 | 否 |
| A6 | 回避核心争点 | 对当事人提出或应当审查的核心争议焦点，裁判文书予以回避、偏移或虚化 | D4,D8,D15 | 0.85 | 是 |
| A7 | 机械复制模板化论证 | 说理部分呈现模板化特征，未针对个案具体情况进行实质性论证 | D8,D9 | 0.65 | 否 |
| A8 | 举证责任倒置异常 | 将本应由一方承担的举证责任不当转移给另一方 | D2,D3,D6 | 0.90 | 是 |

## 分类映射要求
每个检测到的异常点，必须额外标注：
- **A系列分类编号**：映射到A1-A8中最匹配的分类
- **指向获益方**：该异常使哪一方获益（原告/被告/双方/无）
- **反向校验结果**：是否存在对该方有利的对应异常
- **净异常判定**：扣除反向异常后，该异常点是否仍然成立（成立/存疑/不成立）
