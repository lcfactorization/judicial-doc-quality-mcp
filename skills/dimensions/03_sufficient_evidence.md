# 证据确实充分 — 评分标准

## 评估哲学

> [!IMPORTANT]
> 证据是裁判的基石。"证据确实充分"不仅是诉讼法的基本要求，更是司法公正的根本保障。一份优秀的裁判文书，应当对每一项关键证据的采信与否都给出令人信服的理由，让当事人和社会公众理解裁判的事实基础。

**核心认知**：中国司法实践中，证据分析是裁判文书最薄弱的环节之一。常见问题包括：仅罗列证据未分析、未说明采信或不采信理由、对关键证据未置可否、证据链条不完整、举证责任分配不当等。

**参考框架**：
- 证据"三性"审查：真实性、合法性、关联性
- 证据"二力"分析：证据能力（可采性）、证明力（说服力）
- 证明标准分层：刑事案件排除合理怀疑、民事案件高度盖然性、行政案件合法性审查

**民商事专项法律依据**：
- 《最高人民法院关于民事诉讼证据的若干规定》（法释〔2019〕19号，2019年修正）第87-88条（证据审核认定标准）
- 《民事诉讼法》第63-73条（证据种类、举证、质证）
- 《最高人民法院关于适用〈民事诉讼法〉的解释》第90-124条（举证责任、证据审核认定）
- 《民法典》第1230条（环境污染举证倒置）、第1247条（饲养动物举证倒置）
- 《劳动争议调解仲裁法》第6条（用人单位举证责任）

## 评分规则（满分 100 分，起评 100 分，扣分制+加分制并行）

### 一、扣分项

| 编号 | 扣分项 | 扣分值 | 判定标准 |
|:---|:---|:---|:---|
| SE-01 | 关键证据未置可否 | -20 | 对当事人提交的关键证据既未采信也未排除，完全回避 |
| SE-02 | 证据采信缺乏说理 | -15 | 采信或不采信某项证据，但未说明理由 |
| SE-03 | 证据链条不完整 | -15 | 各证据之间缺乏逻辑联系，无法形成完整证据链 |
| SE-04 | 举证责任分配不当 | -18 | 错误分配举证责任，导致裁判结果不公 |
| SE-05 | 证据与事实脱节 | -15 | 认定的事实与采信的证据之间缺乏对应关系 |
| SE-06 | 证据三性分析缺失 | -12 | 未对证据的真实性、合法性、关联性进行分析 |
| SE-07 | 证明力判断无依据 | -10 | 对证据证明力大小的判断缺乏依据 |
| SE-08 | 证据列举不全 | -8 | 遗漏了当事人提交的重要证据 |
| SE-09 | 证据分类混乱 | -5 | 证据分类不清，按当事人罗列而非按证明对象分类 |
| SE-10 | 证明标准把握不当 | -12 | 对不同类型案件适用了错误的证明标准 |

### 二、加分项

| 编号 | 加分项 | 加分值 | 判定标准 |
|:---|:---|:---|:---|
| SE-B01 | 证据三性分析充分 | +5至+8 | 对每项关键证据的真实性、合法性、关联性逐一分析 |
| SE-B02 | 证据链条完整清晰 | +5至+8 | 各证据相互印证，形成完整的证据链条 |
| SE-B03 | 采信/不采信理由充分 | +3至+5 | 对每项证据的采信或不采信都给出了令人信服的理由 |
| SE-B04 | 心证公开 | +3至+5 | 公开法官认定证据的心证过程，说明采信或不采信的内心确信 |
| SE-B05 | 举证责任分配合理 | +3至+5 | 举证责任分配符合法律规定和公平原则 |

### 三、辅助评估框架：证据审查方法论

> [!NOTE]
> 以下框架借鉴自最高法院裁判文书说理方法论研究，用于辅助评估证据分析维度。

**证据审查三步法**：

| 步骤 | 内容 | 评估要点 |
|:---|:---|:---|
| 第一步：证据资格审查 | 审查证据的真实性、合法性、关联性 | 是否逐项审查三性 |
| 第二步：证明力判断 | 比较各证据的证明力大小 | 是否说明证明力判断依据 |
| 第三步：综合认定 | 综合全案证据认定事实 | 证据链条是否完整、逻辑是否严密 |

**证明标准分层**：

| 案件类型 | 证明标准 | 评估要点 |
|:---|:---|:---|
| 刑事案件 | 排除一切合理怀疑 | 是否达到最高证明标准 |
| 民事案件 | 高度盖然性/优势证据 | 是否达到民事证明标准 |
| 行政案件 | 合法性审查+合理性审查 | 是否符合行政证明标准 |

### 五、民商事专项：举证责任分配规则

> [!IMPORTANT]
> 以下为民商事案件举证责任分配的核心规则，评估时应重点审查举证责任分配是否正确。

| 规则 | 内容 | 法律依据 |
|:---|:---|:---|
| "谁主张，谁举证" | 当事人对自己提出的主张有责任提供证据 | 《民事诉讼法》第64条 |
| 举证责任倒置 | 特定侵权案件由被告承担举证责任 | 《民法典》第1230条（环境污染）、第1247条（饲养动物）等 |
| 法院调查收集 | 因客观原因不能自行收集的证据，可申请法院调查收集 | 《民事诉讼法》第64条 |
| 自认规则 | 一方对另一方主张的不利事实明确承认的，另一方无需举证 | 《民事诉讼证据规定》第3条 |
| 推定规则 | 根据已知事实和日常生活经验法则推定另一事实 | 《民事诉讼证据规定》第10条 |

### 六、民商事专项：证明标准分层

> [!NOTE]
> 不同类型的民商事案件适用不同的证明标准，评估时应注意区分。

| 案件类型 | 证明标准 | 说明 |
|:---|:---|:---|
| 一般民事案件 | 高度盖然性/优势证据 | 证据证明力明显大于反驳证据 |
| 侵权案件（举证倒置） | 被告需证明不存在因果关系 | 《民法典》第1230条 |
| 合同案件 | 主张合同成立一方举证 | 《民事诉讼证据规定》第91条 |
| 劳动争议 | 用人单位对解除劳动合同等负举证责任 | 《劳动争议调解仲裁法》第6条 |

### 七、民商事专项：间接证据认定规则

> [!WARNING]
> 民商事案件大量依赖间接证据，以下四条规则是评估间接证据认定是否正确的关键标准。

1. 间接证据之间必须存在印证关系
2. 间接证据必须能够形成完整的证明体系
3. 依据间接证据认定的结论必须具有唯一性或高度盖然性
4. 不得仅凭单一间接证据认定事实

### 八、民商事典型案例（证据采信维度）

> [!NOTE]
> 以下案例摘自《优秀裁判文书选编及点评》，用于辅助评估证据采信维度。

**案例1：钟某诉林某民间借贷纠纷案**——举证责任分配
- 裁判要旨：民间借贷为实践性合同，出借人主张巨额现金交付不符合常理，不予采信。
- 证据采信亮点：合理分配举证责任，强调借贷的实践性特征，对巨额现金交付的主张从严审查

**案例2：A财产保险股份有限公司诉李某某财产保险合同纠纷案**——格式条款说明义务
- 裁判要旨：保险公司未在订立合同时对免责条款履行明确说明义务，免责条款无效。
- 证据采信亮点：对保险法第十八条的适用准确，对说明义务的论证充分
- 专家点评：孙建国（上海市高级人民法院副院长）
