# 说理充分透彻 — 评分标准

## 评估哲学

> [!IMPORTANT]
> 说理是裁判文书的灵魂。一份优秀的裁判文书，不仅在于结论正确，更在于说理过程让当事人心服口服、让社会公众信服。说理充分透彻的文书，能够止诉息讼、化解矛盾；说理空洞的文书，即使结论正确，也难以获得当事人的认同和社会的信任。

**核心认知**：中国司法实践中，说理不充分是裁判文书最普遍的质量问题。最高人民法院多次强调"加强裁判文书说理"，2024年审判质量管理指标体系改革将"一审服判息诉率"列为核心指标，本质上就是要求裁判文书通过充分说理实现案结事了。

**民商事专项法律依据**：
- 《最高人民法院关于加强和规范裁判文书释法说理的指导意见》（法发〔2018〕10号）
- 《民事诉讼法》第152条（判决书内容要求）
- 《最高人民法院关于适用〈民事诉讼法〉的解释》第230条（诉辩归纳与说理）
- 《民法典》第6条（公平原则）、第7条（诚信原则）

## 评分规则（满分 100 分，起评 100 分，扣分制+加分制并行）

### 一、扣分项

| 编号 | 扣分项 | 扣分值 | 判定标准 |
|:---|:---|:---|:---|
| R-01 | 对核心争议焦点完全未回应 | -25 | 文书中存在明确的争议焦点，但"本院认为"部分对该焦点只字未提或仅以一句话带过 |
| R-02 | 说理"贴标签" | -18 | 仅写"于法无据，不予支持""缺乏事实依据，本院不予采信"等结论性语句，未展开任何分析论证 |
| R-03 | 说理逻辑跳跃 | -15 | 从前提直接跳到结论，中间推理环节缺失，读者无法理解从A到B的逻辑链条 |
| R-04 | 说理与判决主文矛盾 | -18 | "本院认为"部分的论证方向与最终判决结果不一致 |
| R-05 | 对当事人重要辩论意见未予回应 | -10 | 当事人在庭审中提出了重要的辩论意见（如时效抗辩、不可抗力抗辩等），文书未予回应 |
| R-06 | 未进行法理分析或价值衡量 | -12 | 对法律适用涉及的价值冲突、利益衡量未作分析，机械适用法条 |
| R-07 | 说理模板化 | -8 | 说理内容为通用模板，未针对个案具体情况展开，可替换为任何同类案件 |
| R-08 | 自由裁量未说明理由 | -10 | 在赔偿数额、责任比例等自由裁量事项上，未说明考量因素和裁量依据 |
| R-09 | 对抗辩意见仅简单否定 | -8 | 对一方当事人的抗辩意见仅写"不成立""不予采纳"，未分析为何不成立 |
| R-10 | 多个争议焦点混为一谈 | -5 | 将多个独立的争议焦点合并论述，导致说理层次不清 |
| R-11 | 三段论推导过程缺失 | -12 | 未明确大前提（法律规范）、小前提（事实）、结论（裁判结果）的推导过程，从事实直接跳到裁判结论 |
| R-12 | 事实推定未说明依据 | -8 | 对事实推定、法律定性、法益衡量等过程未明确说明 |
| R-13 | 对自由裁量权行使未说明裁量依据 | -8 | 在赔偿数额、责任比例等自由裁量事项上，未说明考量因素和裁量依据（与R-08区分：R-08是完全未说明理由，R-13是说明了理由但未说明裁量依据和目的） |

### 二、加分项

| 编号 | 加分项 | 加分值 | 判定标准 |
|:---|:---|:---|:---|
| R-B01 | 争议焦点逐一回应 | +5至+10 | 对全部争议焦点逐一展开论证，每个焦点均有独立的论证段落 |
| R-B02 | 多层次说理 | +5至+8 | 综合运用事理、法理、学理、情理、文理进行多层次论证 |
| R-B03 | 类案参照说理 | +3至+5 | 援引指导性案例或类案检索结果增强说理说服力 |
| R-B04 | 价值衡量公开 | +3至+5 | 对涉及价值冲突的问题，公开进行利益衡量并说明权衡过程 |
| R-B05 | 回应当事人核心关切 | +3至+5 | 不仅回应法律层面的争议，还回应了当事人事实层面的核心关切 |
| R-B06 | 三段论推导完整 | +3至+5 | 明确大前提（法律规范）、小前提（事实认定）、结论（裁判结果），推导过程清晰严密 |
| R-B07 | 情理法融合说理 | +5至+8 | 在合法前提下，兼顾人情、天理、国法，实现"情法两得""情法两平"；对涉及家庭、邻里、伦理等案件注重情理分析 |

### 三、创新性加分项

> [!IMPORTANT]
> 以下创新性加分项旨在鼓励裁判文书在说理层面突破陈规、体现司法智慧。
> 创新性加分与一般加分项叠加计算，但加分后总分不超过100分。
> 创新性加分需同时满足：(1) 有原文引用支撑；(2) 有明确的创新点说明；(3) 符合法律规定和司法伦理。

| 编号 | 加分项 | 加分值 | 判定标准 |
|:---|:---|:---|:---|
| R-IB01 | 调解成功/促成和解 | +5至+10 | 法官通过调解或促成当事人和解，实质性化解矛盾，实现案结事了；或在说理中充分阐释调解方案的法律依据和合理性 |
| R-IB02 | 法律漏洞填补 | +8至+12 | 在法律未有明确规定时，通过法律解释方法（类推适用、目的解释、体系解释等）填补漏洞，创造性解决法律适用疑难问题，并在说理中充分论证解释方法的正当性 |
| R-IB03 | 创造性突破既有框架/打破陈规 | +10至+15 | 打破既有裁判惯例或陈旧框架，在法律允许范围内作出创新性裁判，推动司法进步；说理中明确论证了突破既有框架的必要性和正当性 |
| R-IB04 | 体现司法底层逻辑 | +5至+8 | 裁判说理深刻体现了司法的底层逻辑——公平正义、权利保障、权力制约，而非机械适用法条；说理超越法条文本，触及法律精神和司法本质 |

**创新性加分上限**：创新性加分合计不超过25分，加分后总分不超过100分。

### 四、辅助评估框架：五理说理理论

> [!NOTE]
> 以下框架借鉴自最高法院优秀裁判文书标准研究，用于辅助评估说理充分性维度。

**五理关系**：事理是基础，法理是尺度，学理是辅助，情理是佐料，文理是工具。

| 五理 | 定义 | 优秀标准 | 常见缺陷 |
|:---|:---|:---|:---|
| 事理 | 案件来龙去脉、前因后果 | 事实认定客观、真实、可信 | 事实认定不完整；证据与事实脱节 |
| 法理 | 法律条文、司法解释 | 法律适用正确；法条引用完整规范 | 只引用法条不解释法理；法条引用不规范 |
| 学理 | 法学理论、公理性知识 | 法律规定不明确时引用学理支撑 | 过度依赖学理；学理引用不当 |
| 情理 | 通人情的道理、可接受性 | 涉及人身关系案件体现司法温度 | 情理与法理冲突；缺乏情理考量 |
| 文理 | 语言、形式、技巧 | 说理方式恰当（叙述式/论证式/综合式） | 说理方式单一；语言不精炼 |

### 四、说理方式分类与适用场景

| 说理方式 | 适用场景 | 评估要点 |
|:---|:---|:---|
| 叙述式 | 事实陈述部分 | 事实叙述是否客观、完整、有层次 |
| 总结式 | 无争议事实概括 | 概括是否准确、不遗漏 |
| **论证式** | **争议焦点回应（核心）** | 论证是否严密、逻辑是否连贯、是否回应了全部争议 |
| 解释式 | 法律条文阐释 | 解释是否准确、是否结合案情 |
| 综合式 | 复杂案件的多层次说理 | 是否综合运用多种说理方式 |

### 五、民商事专项：情理法融合的典型场景

> [!NOTE]
> 以下为民商事案件中情理法融合的6类典型场景，评估时应重点关注这些场景下的说理质量。

| 场景类型 | 情理法融合要点 | 典型案件 |
|:---|:---|:---|
| 离婚案件 | 考虑子女利益、家庭贡献、过错程度 | 抚养权纠纷：不仅依法分割财产，更关注子女身心健康 |
| 继承案件 | 尊重遗嘱意愿、照顾弱势继承人 | 法定继承纠纷：对尽主要赡养义务的继承人适当多分 |
| 劳动争议 | 平衡劳资利益、保护劳动者权益 | 违法解除案：既依法认定解除违法，又考虑企业经营实际 |
| 消费者权益 | 保护消费者、规范经营者行为 | 欺诈赔偿案：既支持惩罚性赔偿，又防止权利滥用 |
| 邻里纠纷 | 维护相邻关系、促进和谐共处 | 相邻权纠纷：既依法裁判，又促进邻里和解 |
| 合同纠纷 | 尊重意思自治、维护交易安全 | 格式条款案：既保护弱势方，又维护交易秩序 |

### 六、民商事专项：常见说理逻辑错误

> [!WARNING]
> 以下为民商事案件说理中常见的5种逻辑错误，评估时应重点识别。

| 错误类型 | 表现 | 示例 |
|:---|:---|:---|
| 循环论证 | 用结论证明结论 | "因被告违约，故被告应承担违约责任" |
| 诉诸权威 | 仅引用法条不加论证 | "依照《民法典》第XXX条之规定"（未结合案情） |
| 诉诸情感 | 以情感代替法律论证 | "原告处境令人同情"（但无法律依据支持诉请） |
| 选择性说理 | 只说支持结论的理由，回避反对理由 | 对被告的抗辩意见不予回应 |
| 推理跳跃 | 省略关键推理步骤 | 从"存在资金往来"直接推出"构成借贷关系" |

### 七、民商事典型案例（说理维度）

**案例1：丛某诉上海H置业有限公司商品房预售合同纠纷案**
- 争议焦点：逾期交房违约金调整
- 说理亮点：对违约金过高/过低的认定，不仅考虑法律标准，还考虑房地产市场波动、购房者实际损失等情理因素
- 专家点评：情理法融合的典范

**案例2：金岳军诉上海J物业管理有限公司物业服务合同纠纷案**
- 争议焦点：物业管理瑕疵与物业费减免
- 说理亮点：对物业管理服务标准的界定，既依据合同约定，又考虑行业惯例和实际服务水平
- 专家点评：说理层次分明，从合同约定→行业惯例→实际履行逐层递进
