# 语言精练流畅 — 评分标准

## 评估哲学

> [!NOTE]
> 语言是裁判文书的载体。精练流畅的语言让裁判文书更具可读性和说服力，而冗长混乱的语言则会让当事人和社会公众难以理解裁判理由。本维度权重较低（8%），因为语言问题通常不影响裁判的公正性，但严重影响文书的可读性和权威性。

**核心认知**：中国司法实践中，语言问题主要包括：(1) 冗长重复；(2) 口语化表达；(3) 当事人名称不统一；(4) 法言法语使用不当；(5) 逻辑连接词缺失。其中，冗长重复是最普遍的问题。

**参考框架**：
- **三要素理论**（最高法院优秀课题研究成果）：语言是"基础和血肉"
  - 语体风格：庄重、规范、准确
  - 写作风格：简洁、清晰、有层次
  - 虚词使用规范：如"故"字不加"因为"
- **五理说理理论**：文理是"工具"——语言、形式、技巧

**民商事专项法律依据**：
- 《人民法院民事裁判文书制作规范》（法〔2016〕221号）关于语言文字的规范
- 《民事诉讼法》第152条（判决书内容要求）
- 《最高人民法院关于加强和规范裁判文书释法说理的指导意见》（法发〔2018〕10号）

### 三、辅助评估框架：三要素理论——语言要素

> [!NOTE]
> 以下框架借鉴自最高法院优秀课题研究成果（三要素理论体系），用于辅助评估语言维度。

| 语言要素 | 评估标准 | 常见缺陷 |
|:---|:---|:---|
| 语体风格 | 庄重、规范、准确 | 口语化、随意化 |
| 写作风格 | 简洁、清晰、有层次 | 冗长、混乱、无层次 |
| 虚词使用 | "故"不加"因为"、"因"不加"所以" | 虚词冗余、逻辑连接不当 |
| 法言法语 | 准确使用法律术语 | 术语使用不当、混淆概念 |
| 数字规范 | 金额大写、日期格式统一 | 数字格式混乱 |

### 五、民商事专项：语言规范细则

> [!NOTE]
> 以下为《人民法院民事裁判文书制作规范》关于民商事裁判文书语言的6项规范细则，评估时应参照执行。

| 规范项 | 要求 | 常见错误 |
|:---|:---|:---|
| 当事人名称 | 全文统一使用同一称谓 | 同一当事人前后称呼不一致 |
| 数额表述 | 阿拉伯数字与汉字数字的规范使用 | 金额表述不规范（如"一万块"应为"10,000元"） |
| 时间表述 | 应写明具体日期 | "近日""前不久"等模糊表述 |
| 法律术语 | 使用规范法律术语 | "原告败诉"应为"驳回原告诉讼请求" |
| 引用格式 | 法律名称加书名号，条款号规范 | "合同法58条"应为"《中华人民共和国民法典》第五百五十八条" |
| 简称使用 | 首次使用全称+括号注明简称 | 未定义简称即使用，或简称前后不一致 |

### 六、民商事专项：法言法语常见错误对照

> [!WARNING]
> 以下为民商事裁判文书中常见的法言法语错误，评估LA-13时应重点识别。

| 错误表述 | 正确表述 | 说明 |
|:---|:---|:---|
| "原告败诉" | "驳回原告的诉讼请求" | 败诉非法律术语 |
| "被告赢了这个案子" | "被告的抗辩成立" | 口语化表述 |
| "欠条" | "借条"或"欠条"（区分使用） | 借条证明借贷关系，欠条证明欠款关系 |
| "合同作废" | "合同解除"或"合同无效" | 作废非法律术语，应区分解除与无效 |
| "赔偿精神损失" | "赔偿精神损害抚慰金" | 精神损失非规范表述 |
| "判决如下" | "判决如下"（判决书）/"裁定如下"（裁定书） | 文书类型与判项表述应一致 |
